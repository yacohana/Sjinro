#　SJinro - Skype人狼補助ツール

ほとんどcoffeescriptとjadeです。coffeeもjadeもほぼ初めてなので、正直全部書き直したいレベルの汚さです。ごめんなさい。

##　使い方

Node.js/npm/mongoDBあたりを準備。

`git clone`して、`npm install`して、 grunt使ってないので適当に`coffee -wc .\`とか。おもむろに`node app.js`

## 参考サイト/使用ライブラリ

* express, jade, socket.io, mongoose, passport, passport-local, connect-flash
* angular.js, jQuery, jQuery UI,UI Bootstrap
* Bootstrap, magic(css3)
* [Kikuchy's Second Memory様](http://kikuchy.hatenablog.com/entry/2013/07/03/042221) 認証周り、ほぼコピペさせて頂きました。

* [jquery.redirect.js](http://www.avramovic.info/razno/jquery/redirect/)
* [overlay.js](http://ichi.fool.jp/blog/?p=1576) おれせか様

ありがとうございました。
